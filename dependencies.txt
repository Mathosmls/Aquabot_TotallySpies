# Make sure you have installed all this dependencies before launching our nodes

## ROS 2 packages
sudo apt update
sudo apt install numpy
sudo apt install sensor-msgs
sudo apt install nav-msgs
sudo apt install nav2-map-server
sudo apt install nav2-costmap-2d
sudo apt install nav2-lifecycle-manager
sudo apt install nav2-bringup
sudo apt install std-msgs
sudo apt install geometry-msgs
sudo apt install tf2
sudo apt install tf2-geometry-msgs
sudo apt install libgeographic-dev
sudo apt install libcairo2-dev
sudo apt install ros-<distro>-tf-transformations

## Python dependencies 
pip install numpy
pip install opencv-python
pip install --upgrade transforms3d

N'hésitez pas à nous contacter si vous avez le moindre problème avec le lancement du projet.
